Для запуска открыть index.html.

Исходный код находится в папке js:
	drawFunctions.js - ф-ии отрисовки в странице браузера
	index.js- точка входа, содержит код алгоритмов FF, FFS и переборного (метод назвали bruteForce), а также несколько вспомогательных ф-ий.

Папка styles содержит стили для страницы.
